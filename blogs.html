Question-3: What is event delegation in the DOM?

We apply the event listener to the parent, and when an event happens on any child element, we can detect it

Question-4: What is event bubbling in the DOM?

When you click on an element, the event also affects its parent, then its grandparent, and continues up to the element. This is called event bubbling.

Question-5: How do you create, add, and remove elements using JavaScript?

To create an element, document.createElement("tag"). To add it, parent.appendChild(element) or parent.append(element). To remove it, element.remove().
